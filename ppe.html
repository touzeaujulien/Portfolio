AP 3 : MDS49 - Infrastructure Réseau Complète

2025

Description

Conception et déploiement d'une infrastructure réseau complète pour la Maison Départementale des Sports 49 (MDS49). Mise en place d'une architecture sécurisée et segmentée avec VLAN, Active Directory, supervision, WiFi sécurisé et accès web interne/externe.

Technologies utilisées

  • Cisco Packet Tracer (Routeurs, Switch, AP)
  • Windows Server 2019 (AD, DHCP, DNS, GPO)
  • Debian (Apache2, Zabbix, FreeRadius)
  • VLAN et Routage Inter-VLAN (OSPF)
  • ACL Standards et Étendues
  • NAT/PAT
  • WiFi Sécurisé (WPA2-Enterprise + Radius)

Tâches réalisées

  • Configuration des routeurs Cisco (RDC et MDS49)
  • Mise en place des ACL de filtrage inter-VLAN
  • Configuration du switch avec VLAN (10, 20, 30, 100)
  • Installation et configuration de la borne WiFi Cisco
  • Déploiement du serveur DHCP avec réservations
  • Configuration du NAT/PAT pour l'accès web externe
  • Création des plans de test (VLAN, DHCP, NAT, Web)
  • Supervision réseau avec Zabbix
